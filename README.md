# CharlieHR

The solution code can be found in `solution.js`, while the test code is in `solution.test.js`.

### Install with npm

In order to run the test script, please clone this repository, navigate to the root and `npm install` to install the test modules. Then run `npm run test`. A suite of 6 tests should run. 

### Install with Yarn

In order to run the test script, please clone this repository, navigate to the root and `yarn install` to install the test modules. Then run `yarn run test`. A suite of 6 tests should run.
